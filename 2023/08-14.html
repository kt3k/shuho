Week 33 - 2023

The weeknote for 8/14 - 8/20.

OSS

Deno KV Connect

  • Deno added KV Connect protocol. Now everybody can create backend of Deno KV by implementing the protocol. Deno KV looks getting more and more interesting!

Node.js

  • Node.js now has the builtin support of --env option. When it's specified the runtime read env variables from the specified file. This looked very surprising move to me.

    Node.js used to have 'small core' design principle, but apparently it doesn't anymore. Node.js looks to have completely changed its design process.

minurl

https://github.com/kt3k/minurl

  • Created a minimalistic url shortner with Deno and Deno KV. It's incredible that we can write a url shortner in around 20 lines of code.

Tech

Terraform

  • Terraform changed its license and now potential competitor can't use it for free. What a terrible move!

    On the other hand, OpenTF (now OpenTofu) has been announced. I definitely support this move.

Tech

ReadableStreamBYOBReader

  • This week I learned about ReadableStreamBYOBReader. I used to be wondering why this was named like this, but now I finally understood it. This reader accepts ArrayBuffer as parameter and fill bytes in it. This is very similar to Go's reader interface.

    This is useful when you want to read less than the exact length of bytes.

genhtml

  • I keep using genhtml CLI tool for creating html coverage report from lcov file. This week I found this tool is a 10K+ line perl script. It feels fantastic that someone still maintain a useful tool written in perl in 2023.

HTTP methods

  • HTTP is now defined in RFC 9110. So we probably naturally assume that what an HTTP method is is defined in that document. Interestingly in deno_std, we couldn't reach consensus on what is a HTTP method. In other words, we couldn't reach consensus on what verbs should be allowed for HTTP method. So we deleted method module from std/http of Deno.

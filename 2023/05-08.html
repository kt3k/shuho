Job

GitHub API

  • Was interacting GitHub API this week. I found a bug with the endpoint "Create a repository using a template". When I was calling this API, the call never succeeded and the error message was unintelligible. After a number of tries I finally reached out to the support. After a few communications, we found that the access permission is required for the template repository itself (whic should have been unnecessary when the template repo is public). We ended up using a little complicated workaround to it.

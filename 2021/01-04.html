Tech

React Server Component

  • I first read a little seriously about React Server Component, and I found I'm not interested in this technology. This is server side rendering technology with more (allegedly) practical and fine-grained features.

    I don't like this feature because of 2 reasons. First, it has thier own module resolution. The custom module resolution means your code is locked-in to the bundler. So your code only works with React Server Component-aware bundlers, and not an universal React component anymore. Second, I don't like the entire SSR technologies. I love jamstack. I believe jamstack is the future. React Server Component is heading completely for the opposite direction. So this is something I definitely avoid at least for now.

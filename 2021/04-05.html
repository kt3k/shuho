Week 14 - 2021

The weeknote for 4/5 - 4/11.

OSS

denoland/deno

  • This week I was working on the cache of CI of denoland/deno. I introduced mtime-restoring python script. It worked on some cases, but didn't work on some cases. The root problem was that mtime of files in build assets and mtime restored by the script sometimes have conflict. Bert created more regorous mtime-restoring script in node.js as github action which saves mtime from the actual mtime when the cache is created and restores mtime based on git object hash (#10110). That solution totally solved the CI cache issue.

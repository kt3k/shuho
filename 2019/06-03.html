‚ìÇÔ∏è saku is a markdown based task runner.

  • This week I created the script called saku-wrapper ( https://github.com/kt3k/saku-wrapper ), which is the wrapper script for my tool saku. This is inspired by the wrapper scripts gradle-wrapper and maven-wrapper, which are wrapper scripts for gradle and maven respectively. If you put saku-wrapper in your repository, the contributors don't need to install saku on their own, but saku-wrapper take care of downloading it and invoking it, just as gradle-wrapper does for gradle.
